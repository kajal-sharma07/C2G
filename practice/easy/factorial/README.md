# Factorial of N

## Difficulty: Easy

## Description
### Description
Write a program that takes a non-negative integer **N** and prints its factorial ($N!$).
The factorial of $N$ is the product of all positive integers less than or equal to $N$. Note that $0! = 1$.

### Input Format
A single integer **N**.

### Output Format
Print the factorial of **N**.

### Constraints
* $0 \\le N \\le 12$

### Examples
**Example 1:**
Input:
```
0
```
Output:
```
1
```

**Example 2:**
Input:
```
5
```
Output:
```
120
```
